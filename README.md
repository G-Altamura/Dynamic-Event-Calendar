# Dynamic-Event-Calendar
A monthly event planner with an interactive calendar interface. Users can add, edit, and delete events, navigate between months, and view daily events. 
Designed as a front-end exercise focused on interactivity and responsive layout.


# Project Description  
Create a monthly event planner with an interactive calendar view.  
Users can add, edit, and delete events.

# Functional Requirements  
1. Calendar View 7×6 day grid with current day highlighting  
2. Add Event Modal with date, time, title, and description  
3. Event Management Click on a day to view/delete events  
4. Navigation Previous/next month arrows  
5. Responsive Single-column layout on mobile
